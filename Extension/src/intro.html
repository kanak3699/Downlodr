Introduction:

Downloadr is a chrome based extension that helps users to download videos/audios from websites such as youtube. Also it provides users to select the desired format such as .mp3 or .mp4 and various others. Moreover, it also provides users to name the file as per their need and the design of the extension is user friendly which helps the user to use it in an efficient way.

Essential Steps:

Setting up extension on Google Chrome:

  1. Navigate to "chrome://extensions/" on Google Chrome.
  2. Select "Load Unpacked" button on the top left corner of Screen.
  3. Navigate through the "src" folder of the file provided
  4. Select the file "popup.html" and click enter.

Running on server:

  1. Navigate to "https://nodejs.org/en/download/" and download the Nodejs file suitable for your operating system.
  2. Open a terminal window on MacOs or Command Prompt on windows and navigate to the src folder.
  3. Run "npm install" command.
  4. Run "node index.js" command to start the server.

Steps to downlad a video/audio file:

  1. Click on "Collect Videos" to collect the video/audio links which are listed on the webpage and select the desired link.
  2. Enter your desired file name.
  3. Click the download button. Your file will be downloaded under the folder named "DownlodrExtension".
  4. To download video file from Youtube then you need first run the server as specified under "Running on Server".
  5. Then open the browser and type "localhost:"listening port (for eg: 3000)""
  6. Open youtube and then follow the steps 1 to step 3 to download the videos.
